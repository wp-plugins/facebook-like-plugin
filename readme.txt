The Facebook Like plugin adds the Facebook Like Button to the end of your blog posts.

== Description ==

This plugin ads the Facebook Like Button to the end of each of your posts and pages.
This functionality easily allows your friends to promote your posts on their FB
profiles.

== Installation ==

1. Download facebook-like-button-plugin.zip
2. Unpack and upload file ‘facebook-like-button-plugin.php’ to ‘/wp-content/plugins/’.
3. Head to your dashboard and activate the plugin.
4. That is all, the plugin is ready to go!

== Frequently Asked Questions ==

= Why use this plugin? =

This plugin automatically ads the FB like button so you don't have to manually do it for each post.
